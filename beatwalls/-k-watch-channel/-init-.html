KWatchChannel(file: File, scope: CoroutineScope = GlobalScope, mode: Mode, tag: Any? = null, channel: Channel<KWatchEvent> = Channel())

Channel based wrapper for Java's WatchService

Parameters

file -

  • file or directory that is supposed to be monitored by WatchService

scope -

  • CoroutineScope in within which Channel's sending loop will be running

mode -

  • channel can work in one of the three modes: watching a single file, watching a single directory or watching directory tree recursively

tag -

  • any kind of data that should be associated with this channel, optional
