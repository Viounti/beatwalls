asWatchChannel

@ExperimentalCoroutinesApi fun File.asWatchChannel(mode: Mode? = null, tag: Any? = null, scope: CoroutineScope = GlobalScope): KWatchChannel

Watches directory. If file is supplied it will use parent directory. If it's an intent to watch just file, developers must filter for the file related events themselves.

Parameters

mode -

  • mode in which we should observe changes, can be SingleFile, SingleDirectory, Recursive

tag -

  • any kind of data that should be associated with this channel

scope -

  • coroutine context for the channel, optional
